Hypothesis Registry

Every claim the lab tracks — its standing on the 0–6 evidence ladder, where it's modeled, and the decisive experiment that would advance or falsify it. Nothing here sits above Level 2 (simulation candidate); evidence is simulation only until a real lab weighs in.

Conducted research

The base tools are sterile by default; use Load into models to populate them from a result and build further. Nothing here changes the underlying model, and nothing leaves your machine.

Autonomous triage

Lab-Scientist Loop

A bounded, self-driving loop that proposes research avenues, runs the simulated screens itself, triages them honestly, and acts on its own suggestions — while never conflating “survived triage” with “validated.” It runs in the Python package (orme_lab.lab_loop); a real sample digest is shown below.

The split that keeps it honest

A creative generator (the orme-lab-scientist subagent) proposes avenues; a deterministic core judges them. They never swap roles — an LLM ranking its own conjectures is idea-triage, not evidence.

Anti-tautology gate

An avenue is a finding only if its predictors reference an off-gate signal — something not re-derivable from the AND-gate’s own inputs. Otherwise it’s flagged TAUTOLOGICAL before it can score.

Must be able to fail

Retiring a hypothesis is a success. Every avenue carries a falsification condition that can fire; a verdict of KILLED_HYPOTHESIS is progress, not failure.

Honesty by type

The verdict vocabulary is KILLED_HYPOTHESIS / SURVIVED / TAUTOLOGICAL / INCONCLUSIVE — there is no “VALIDATED” member. Tier-3 “invent a new mechanism” avenues are quarantined, never run. Everything is clamped to Level 2.

Scoped hypotheses

Some claims are element- or geometry-specific, so the loop splits them so they retire independently: H1-open-shell {Ir, Os, Pt, Rh, Ru} vs H1-closed-shell {Ag, Au, Pd}, and H3-cluster vs H3-monomer. The boundary is the real d_shell_vacancies property, which exactly predicts the toy anisotropy — so killing the closed-shell claim never touches the open-shell one.
